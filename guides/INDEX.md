**Q: How do I start?**
A: Read QUICK_START.md (15 min), then run `python src/screen_recorder.py`

**Q: Where are videos saved?**
A: `d:\Desktop\screenshare\recordings\` folder

**Q: Can I blur faces?**
A: Yes! Check "ğŸ”’ Blur Faces" before recording. See FACE_BLUR_FEATURE.md

**Q: What if it's slow?**
A: Close other apps, reduce resolution/FPS, or disable face blur. See README.md

**Q: How do I customize blur?**
A: Edit BLUR_SIGMA and BLUR_KSIZE in src/screen_recorder.py. See FACE_BLUR_FEATURE.md

**Q: Can I record audio?**
A: No, video only. Use OBS Studio for audio+video. See QUICK_START.md

A: Python 3.7+, 2GB RAM, Windows/Mac/Linux. See README.md
